## 🛠️ 包含的工具

### 🎵 krr AnyKeys 转谱器
将 osu!mania 谱面转换至任意轨道数。

### 📝 凉雨转面器
改变面条占比以及面条长度。

### 🗝 DP 工具
使轨道数翻倍，并可调密度。

### 🗑️ .osu 谱面文件管理
整理亦或是移除多余的 .osu 单谱。

### 📊 krr LV 评级系统
自动以 KRRLV 给谱面评级。
